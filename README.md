# a2b

Shows an ascii tree in the terminal or outputs to JSON when given an array representing a binary tree.

## Install

```
npm install a2b
```

or 
```
yarn add a2b
```


## Usage

```
  $ a2b <json_array>
```

## Options

```
  --json, -j  Output tree to JSON string
```

## Examples

```
$ a2b [1,2,3]
    1
  /   \\
 /     \\
2       3

```

## Build and Run

Builds and compiles to dist/

```
yarn build
```
